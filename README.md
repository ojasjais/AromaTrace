# 🌿 AromaTrace

A modern full-stack SaaS application for managing and tracking essential oil production batches.

Built using React, Node.js, Express, Prisma, PostgreSQL (Supabase), and Tailwind CSS.

---

## ✨ Features

- 🌿 Premium SaaS Landing Page
- 📊 Analytics Dashboard
- 📦 Batch Management
- ➕ Create Batch
- ✏️ Edit Batch
- ❌ Delete Batch
- 🔍 Search & Filter
- 📈 Production Statistics
- 🌙 Dark Mode
- 📱 Fully Responsive
- ⚡ Glassmorphism UI
- 🎨 Framer Motion Animations
- 🗄 PostgreSQL Database (Supabase)
- 🔐 REST API using Express & Prisma

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/ojasjais/AromaTrace
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```

## 🔑 Environment Variables

Backend `.env`

```env
DATABASE_URL=
```
